# Numpy VS Cupy

CPU vs GPU performance

### Pre-requisitos:

- Python3.11
- pip

### Comandos:

1 - `pip install poetry`

1 - `poetry install`

2 - `poetry shell`

3 - `chmod +x run_cpu.sh run_gpu.sh`

4 - `./run_cpu.sh`

4 - `./run_gpu.sh`
